# Лабораторна робота №3: CLI програма на Node.js

**Варіант 5**: Робота з даними про споживання пального та метриками авто (`mtcars.json`).

## Опис
Програма працює через командний рядок та дозволяє читати JSON-файл з даними про автомобілі, фільтрувати їх за певними критеріями та виводити результат у консоль або зберігати у новий файл.

## Використання
Програма приймає наступні параметри:

**Загальні параметри:**
* `-i, --input <шлях>` : (Обов'язковий) Шлях до вхідного файлу `mtcars.json`.
* `-o, --output <шлях>`: (Необов'язковий) Шлях до файлу для збереження результату.
* `-d, --display`      : (Необов'язковий) Вивести результат у консоль.

**Параметри фільтрації:**
* `-c, --cylinders <число>`: Відображати кількість циліндрів (поле `cyl`).
* `-m, --mpg <число>`      : Відображати лише записи машин з паливною економністю нижче за зазначену (поле `mpg`).

## Приклади запуску:
1. Тільки вивід у консоль:
   `node index.js -i mtcars.json -d`
2. Фільтрація авто (наприклад, менше 20 mpg) і запис у файл:
   `node index.js -i mtcars.json -m 20 -o result.json`
